This teacher and student communication tool.

Three types of user in this application.

user : Students

Student can as question on any
subject. Considering that student may have many subject to
learn. Student will ask question or raise his doubt and save it

admin : Teachers or admin.

user will answer the question and if feel that some image/picture
is require to upload he can upload a picture. right now we are
uploading a single picture. We can enhance this function under
the discussion link of answer. Teacher will answer the question

super : Super / Principal.

this user can access the whole application. all links and tab are accessible to super user.
super user can check the status of teacher and students. He can monitor how question
are responded by teacher.
